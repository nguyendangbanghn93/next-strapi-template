# app landing page

## How to run **app**

- Install Mysql
- In `/app/backend` run `yarn`
- In `/app/frontend` run `yarn`
- In `/app`, create .env file, edit config then run `yarn  develop`

*Example:*

```bash
cd ./app/backend
npm install
```
```bash
cd ./app/backend
npm install
```
```bash
cd ./app
npm install
npm run develop
```


- Go to backend `http://localhost:1337/admin`
- Go to frontend `http://localhost:3000`
